# Student Attendance System

A complete Student Attendance System with MVC architecture using Node.js, Express, MongoDB, and EJS with email notifications and animated UI.

## Features

- **Students Management**: Add, view, edit, and delete student records
- **Attendance Tracking**: Mark, view, update, and delete attendance records
- **Email Notifications**: Automatic email alerts when attendance is marked or updated
- **Animated UI**: Interactive and responsive user interface with animations
- **MVC Architecture**: Organized using Models, Views, Controllers, and Routes

## Tech Stack

- **Backend**: Node.js, Express.js, MongoDB (with Mongoose)
- **Frontend**: EJS templates, CSS3, Vanilla JavaScript
- **Email**: Nodemailer
- **Icons**: Feather Icons

## Project Structure

The project follows an MVC (Model-View-Controller) architecture:

- **Models**: Define MongoDB schemas and data structure
- **Views**: EJS templates for rendering pages
- **Controllers**: Handle business logic and request handling
- **Routes**: Define endpoints and HTTP methods
- **Public**: Contains static assets (CSS, JavaScript)
- **Utils**: Utilities, such as the email mailer
